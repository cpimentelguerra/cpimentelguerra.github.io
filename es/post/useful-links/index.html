Links de interés

Última actualización el 18 Mar, 2022 2 min de lectura

En esta sección he recogido diferentes programas, páginas web y otros recursos que utilizo diariamente en mi trabajo. Además, también he añadido otros enlaces a programas, páginas web y apps que considero interesantes. Esta sección se actualiza constantemente según voy encontrando nuevas utilidades.

Programas científicos

Gestor de referencias

Zotero: En mi opinión, el mejor gestor de referencias (para Windows, Mac and Linux).

Zoo for Zotero: App de Zotero para Android.

Zotero iOS: App oficial de Zotero para iPhone y iPad.


Cálculos de geoquímica acuosa

PHREEQC notepad++: La mejor versión para trabajar con PHREEQC cuando tienes experiencia.

PHREEQCi: La mejor versión de PHREEQC para aprender.

PHREEQC página web

PHREEQC Users (foro)


Visualización de estructuras cristalinas

VESTA

VESTA tutorial: Tutorial para visualizar estructuras cristalinas en VESTA.


Análisis difractogramas de polvo

Profex

Profex canal de youtube: Tutoriales, consejos y revisiones de la actualizaciones de Profex.



Páginas web

Bases de datos de minerales

Mindat: La base de datos de minerales más completa.

Webmineral: Página web desactualizada. Sin embargo, tiene una herramienta de búsqueda muy interesante que permite buscar los minerales por los picos de difracción.

RRUFF: Base de datos de minerales que incluye espectros Raman y difracción de rayos X.


Bases de datos de estructuras cristalinas

American Mineralogist crystal structure database

Crystallography Open Database


Modelos cristalinos en 3D

Tarugos 3D: Morfologías de cristales y estructuras para imprimir en 3D.

Smorf crystals: Morfologías de cristales para visualizar en 3D (incluye la indexacción de las caras).


LaTeX

Overleaf: Editor de LaTeX online.


Varios

EPFL Crsytallography: Página web del EPFL con numerosas aplicaciones java para cristalografía.



Otros programas

3D software

FreeCAD: Open Source CAD.

Blender: Programa de modelado 3D Open Source.

3DF Zephyr: Programa de fotogrametría fácil de usar.

Meshroom: Programa de fotogrametría Open Source.



Apps

XteroApp: App para practicar la proyección estereográfica de modelos cristalográficos.
